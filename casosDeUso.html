API

Los administradores, los usuarios, los compradores y los vendedores pueden ingresar a la aplicación

    • URL: '/identificacion/AIdentificar'
    • Método: POST
    • Formato de entrada y salida: JSON
    • Parámetros POST:
      • correo - Requerido. Long. mínima: 1.
      • clave - Requerido. Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: No disponible

    Cursos posibles

    1. resultado:

      • label:'/VPerfil',
      • msg:"Usuario Identificado".

      Se continúa con el rol del usuario.

    2. resultado:

      • label:''/identificacion/AIdentificar',
      • msg:"Usuario no se pudo identificar".

    • URL: '/inmuebles/VInmuebles'
    • Método: GET
    • Formato de entrada y salida: JSON
    • Roles: Sin restricción de rol
    • Requisitos: No disponible
    • Resultados: se devuelven todos los inmuebles ofertados que cumplan con filtros de búsqueda

    • URL: '/Inmuebles/VDetalles/id_inmueble'
    • Método: GET
    • Formato de entrada y salida: JSON
    • Parámetros GET:
      • id_inmueble
      • Requerido.
      • Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: No disponible
    • Resultados:
      • id_inmueble
      • descripción
      • precio
      • dirección
      • áreaConstrucción
      • áreaTerreno
      • númeroHabitaciones
      • númeroBaños
      • númeroEstacionamientos
      • tipoInmueble
      • fotoPrincipal
      • fotosSecundarias

    • URL: '/inmuebles/AHacerOfertaDeCompra/'
    • Método: POST
    • Formato de entrada y salida: JSON
    • Parámetros POST:
      • correo - Requerido. Long. mínima: 1.
      • id_inmueble - Requerido. Long. mínima: 1.
    • Roles: Cliente
    • Requisitos: Usuario registrado con sesión iniciada.

    Cursos posibles

    1. resultado:

      • label:'/inmuebles/VDetalles/id_inmueble',
      • msg:"Oferta de Compra Hecha".

      Se continúa con el rol del usuario.

    2. resultado:

      • label:'/inmuebles/VDetalles/id_inmueble',
      • msg:"No se pudo hacer la oferta de compra".

    • URL: '/identificacion/ARegistrar'
    • Método: POST
    • Formato de entrada y salida: JSON
    • Parámetros POST:
      • correo - Requerido. Long. mínima: 1.
      • clave - Requerido. Long. mínima: 1.
      • nombres - Requerido. Long. mínima: 1.
      • apellidos - Requerido. Long. mínima: 1.
      • númeroDeCédula - Requerido. Long. mínima: 1.
      • dirección - Requerido. Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: No disponible

    Cursos posibles

    1. resultado:

      • label:'/VPerfil',
      • msg:"Usuario Registrado".

      Se continúa con el rol del usuario.

    2. resultado:

      • label:''/identificacion/ARegistrar',
      • msg:"Usuario no se pudo registrar".

    • URL: '/inmuebles/AHacerOfertaDeVenta'
    • Método: POST
    • Formato de entrada y salida: JSON
    • Parámetros POST:
      • correo: Requerido
      • descripción: Requerido
      • precio: Requerido
      • dirección: Requerido
      • áreaConstrucción: Requerido
      • áreaTerreno: Requerido
      • númeroHabitaciones: Requerido
      • númeroBaños: Requerido
      • númeroEstacionamientos: Requerido
      • tipoInmueble: Requerido
      • fotoPrincipal: Requerido
      • fotosSecundarias: Opcional
    • Roles: Vendedor
    • Requisitos: Usuario registrado con sesión iniciada.

    Cursos posibles

    1. resultado:

      • label:'/Inmuebles/VDetalles/id_inmueble',
      • msg:"Oferta de Venta Hecha".

      Se continúa con el rol del usuario.

    2. resultado:

      • label:'/inmuebles/AHacerOfertaDeVenta',
      • msg:"No se pudo hacer la oferta de venta".
