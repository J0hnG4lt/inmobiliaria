API

Los administradores, los usuarios, los compradores y los vendedores pueden ingresar a la aplicación

    • URL: '/identificacion/AIdentificar'
    • Método: POST
    • Formato de entrada y salida: JSON
    • Parámetros POST:
      • correo - Requerido. Long. mínima: 1.
      • clave - Requerido. Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: No disponible

    Cursos posibles

    1. resultado:

      • label:'/VPerfil',
      • msg:"Usuario Identificado".

      Se continúa con el rol del usuario.

    2. resultado:

      • label:''/identificacion/AIdentificar',
      • msg:"Usuario no se pudo identificar".

    • URL: '/inmuebles/VInmuebles'
    • Método: GET
    • Formato de entrada y salida: JSON
    • Parámetros GET:
      • número_de_página
      • Requerido.
      • Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: No disponible

    • resultado:

      • inmuebles: lista paginada de objetos de tipo inmueble que todavía estén en venta. Si el usuario es administrador, entonces puede ver todos las ofertas de inmuebles

    • URL: '/VNosotros'
    • Método: GET
    • Formato de entrada y salida: JSON
    • Roles: Sin restricción de rol
    • Requisitos: No disponible
    • Resultados: se devuelve la información pertinente de la compañía

    • URL: '/VOficinas'
    • Método: GET

    • Resultados: se devuelve la información pertinente a las oficinas

    • URL: '/VPerfil'
    • Método: GET
    • Formato de entrada y salida: JSON
    • Parámetros GET:

      • Requerido.
      • Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: el usuario fue el creador de ese perfil

      • nombres
      • apellidos

      • teléfono

    • Método: GET

    • Resultados: se devuelve la información de contacto de la compañía

    • URL: '/Inmuebles/VDetalles/id_inmueble'
    • Método: GET
    • Formato de entrada y salida: JSON
    • Parámetros GET:
      • id_inmueble
      • Requerido.
      • Long. mínima: 1.
    • Roles: Sin restricción de rol
    • Requisitos: No disponible
    • Resultados:
      • id_inmueble (Requerido)
      • precio (Requerido)
      • descripción (Requerido)
        • tipoDeInmueble
        • tipoDeOperación
        • antiguedadDelInmueble
        • númeroDeHabitaciones
        • númeroDeBaños
        • nombreDelInmueble
        • NúmeroDeEstacionamientos
        • metrosDeConstrucción
        • metrosDeTerreno
        • dirección
      • característicasAdicionales (Opcional)
        • Lista personalizable de tamaño variable de características
      • instalacionesComodidades (Opcional)
        • Lista personalizable de tamaño variable de instalaciones
      • cercanías (Opcional)
        • Lista personalizable de tamaño variable de cercanías
      • detalles (Opcional)
        • Lista personalizable de tamaño variable de detalles
      • horarioNegociación (Requerido)
        • horaContactoInicio
        • horaContactoFin
      • fotoPrincipalURL (Requerido)
      • fotosSecundarias (Requerido)
        • Lista personalizable de tamaño variable de fotos
      • fechaDeOferta (Requerido)
      • sePuedeMostrar (Requerido) Rol: Administrador

    • URL: '/inmuebles/AHacerOfertaDeCompra/'
    • Método: POST
    • Formato de entrada y salida: JSON
    • Parámetros POST:
      • correo - Requerido. Long. mínima: 1.
      • id_inmueble - Requerido. Long. mínima: 1.
    • Roles: Cliente
    • Requisitos: Usuario registrado con sesión iniciada.

    Cursos posibles

    1. resultado:

      • label:'/inmuebles/VDetalles/id_inmueble',
      • msg:"Oferta de Compra Hecha".

      Se continúa con el rol del usuario.

    2. resultado:

      • label:'/inmuebles/VDetalles/id_inmueble',
      • msg:"No se pudo hacer la oferta de compra".
